## About This Project

**AI Disease & Symptom Analysis Assistant** is a machine learning project designed to help users explore possible health conditions based on their reported symptoms using natural language.

The system is powered by a fine-tuned **DistilBERT transformer model** trained on a disease and symptom dataset. Unlike traditional symptom checkers that require users to select predefined symptoms, this application allows users to describe how they feel in their own words. The model analyzes the provided description and returns possible matching conditions with confidence scores.

Click [here](https://drive.google.com/drive/u/2/folders/1yJ52yAmXVILWempV7Ut4AHstcuXvlsGp) to view model weights

The goal of this project is to demonstrate how modern Natural Language Processing (NLP) techniques can be applied to healthcare-related text understanding and provide an accessible way to explore symptom patterns.

### How It Works
- Users enter a free-text description of their symptoms.
- The text is processed using a transformer-based NLP model.
- The model predicts the most likely matching conditions from the diseases it was trained on.
- Results are presented as possible matches with confidence estimates.

## Health Disclaimer

⚠️ **Important: This application is not a medical diagnosis tool.**

The predictions provided by this system are generated by an artificial intelligence model and are intended for **educational and informational purposes only**. They should not be used as a substitute for professional medical advice, diagnosis, or treatment.

Symptoms can vary widely between individuals, and the same symptoms may be associated with multiple conditions. If you are experiencing concerning, severe, or persistent symptoms, please consult a qualified healthcare professional.

Do not use this tool for emergency medical situations. If you believe you are experiencing a medical emergency, seek immediate medical assistance.
